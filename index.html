Skill Toolbelt

Operating Systems

Windows, Linux, and macOS

Security Principles and Practices

Familiarity with security concepts, principles, and best practices such as confidentiality, integrity, availability, risk assessment, access controls, encryption, and security frameworks like NIST, ISO 27001, or CIS Controls

Networking

A solid understanding of networking protocols, TCP/IP, subnetting, firewalls, routers, and switches is essential for analyzing network traffic, identifying vulnerabilities, and implementing security measures.

Threat Landscape Awareness

Knowledge of different types of cyberattacks, malware, social engineering techniques, phishing, ransomware, and other common attack vectors.

Security Technologies

Intrusion detection/prevention systems (IDS/IPS), antivirus software, virtual private networks (VPNs), authentication mechanisms

Incident Response

Knowledge of incident response processes, including identification, containment, eradication, and recovery from security incidents, as well as the ability to investigate and document security breaches.

Security Tools and Software

Familiarity with security tools such as Wireshark, Nmap, Metasploit, Burp Suite, Snort, Splunk, MySQL, Git, SIEM (Security Information and Event Management), and other tools commonly used in cybersecurity operations.
