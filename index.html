Projects

From most recent to oldest, here are prominent projects I've done. The projects with * indicate they are a work-in-progress project.

  • As a way to volunteer my services to my local library, I gathered, organized, and analyzed Los Angeles County Library System to perform Resource Allocation Analysis. Check out the project requirements. *
  • I built a typewriter simulator which is a text editor where you can't use the backspace. Try it out here: Typewriter app
  • Through a guided project written by Marynia Kolak, Moksha Menghaney, Qinyun Lin, and Angela Li, I learned how to perform geospatial analysis techniques such as: geocoding, buffer analysis, and nearest resource analysis. GitHub Repo.
  • In part of completing the Google Data Analytics Certificate program, I created a capstone project to analyze Greenhouse Gas Emissions data from the Environmental Protection Agency. Find the full report here: RPubs
  • I analyzed AM Wedding Film Co. Facebook Ads data to identify and track Key Performance Identifiers and then based on the data insights, recommended actionable solutions that reduced marketing expenses by 5%.
  • The game Elden Ring exploded in popularity in 2022 and many Reddit users had negative and positive thoughts about its game mechanics. So using Reddit's API, I parsed and cleaned qualitative data to conduct sentiment analysis to discover the impact of Elden Ring's boss mechanics. Find the tutorial here: Elden Ring Sentiment Analysis Tutorial
  • I created an automated Twitter account that requests Twitter's API to tweet daily inspirational quotes. Find the account here: @SunshinecatLeo and build your own automated account with my tutorial
